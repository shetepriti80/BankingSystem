# Banking System Project

A simple Java-based Banking System developed using JDBC and MySQL.  
It provides basic banking operations such as account creation, deposit, withdrawal, balance enquiry, and account management through a console-based interface.

---

## 📌 Features
- Create new bank account  
- Deposit money  
- Withdraw money  
- Check account balance  
- View account details  
- Delete account  

---

## 🛠️ Technologies Used
- Java  
- JDBC  
- MySQL  
- Eclipse / Command Line Interface
